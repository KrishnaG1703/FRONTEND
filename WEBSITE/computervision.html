COMPUTER VISION

HAND TRACKING

I have developed a hand tracking project using Python, employing computer vision techniques and the Mediapipe library. This project enables real-time detection and tracking of hand movements and gestures through a webcam feed. By leveraging computer vision algorithms provided by Mediapipe, it accurately identifies key landmarks on the hand, facilitating various applications such as gesture recognition, virtual touch interfaces, and augmented reality interactions.

FACE DETECTION

I have developed a face detection project using Python, leveraging the capabilities of computer vision and the MediaPipe library. This project utilizes advanced algorithms to accurately detect and locate faces within images or video streams. Through the integration of MediaPipe's powerful features, including facial landmark detection and tracking, the project provides real-time feedback on detected faces, opening up possibilities for various applications such as facial recognition, emotion analysis, and augmented reality experiences.

POSE ESTIMATION

I have developed a pose estimation project using Python, employing computer vision techniques and leveraging the capabilities of the MediaPipe library. This project enables accurate real-time detection and tracking of human body poses from images or video streams. By utilizing advanced algorithms, it provides valuable insights into human movement and posture, with potential applications ranging from fitness tracking and sports analysis to gesture recognition and augmented reality interactions.
